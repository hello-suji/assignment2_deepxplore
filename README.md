# Assignment 2 - DeepXplore 스타일 차등 테스트

## 1. 개요

본 프로젝트는 DeepXplore의 아이디어를 기반으로 딥러닝 모델에 대한 차등 테스트(differential testing)를 수행한다.
CIFAR-10 데이터셋으로 학습된 두 개의 ResNet50 모델을 사용하여 서로 다른 예측을 수행하는 입력을 탐지하고, 뉴런 커버리지(neuron coverage)를 측정한다.

본 실험의 목적은 동일한 입력에 대해 서로 다른 예측을 생성하는 경우를 탐지함으로써, 모델의 오류 가능성이 있는 입력이나 경계 사례를 식별하는 것이다.

## 4. 모델 구성

* 모델 구조: ResNet50
* 데이터셋: CIFAR-10
* 모델 개수: 2개

각 모델은 서로 다른 설정으로 독립적으로 학습되었다.

| 모델      | 랜덤 시드 | 학습률    |
| ------- | ----- | ------ |
| Model A | 42    | 0.001  |
| Model B | 123   | 0.0005 |

또한 CIFAR-10에 맞게 ResNet50의 마지막 fully connected layer를
1000 클래스에서 10 클래스로 수정하였다.

---

## 5. 구현 세부 사항 (DeepXplore 적용)

본 구현은 DeepXplore의 핵심 개념을 다음과 같이 적용하였다.

### 5.1 차등 테스트

* 동일한 입력을 두 모델에 동시에 입력
* 서로 다른 예측을 생성하는 입력을 수집
* 이러한 입력을 오류 가능성이 있는 사례로 간주

---

### 5.2 뉴런 커버리지

* ReLU 활성값을 기반으로 커버리지 측정
* 특정 임계값 이상으로 활성화된 뉴런을 “커버됨”으로 간주
* 커버리지는 다음과 같이 계산된다.

```
(활성화된 뉴런 수) / (전체 뉴런 수)
```

* 단순한 방식이므로 커버리지는 빠르게 1.0에 도달할 수 있음

---

### 5.3 주요 수정 사항

* CIFAR-10 이미지를 ResNet50 입력 크기에 맞게 리사이즈
* 출력 레이어를 1000 → 10 클래스로 수정
* forward hook을 이용한 뉴런 커버리지 직접 구현
* ResNet 구조에서 발생하는 activation shape 문제 해결 (shape 기반 key 분리)
* DeepXplore 원본 코드 대신 PyTorch 기반으로 직접 구현
